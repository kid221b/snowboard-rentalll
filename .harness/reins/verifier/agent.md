# Verifier Agent

Adversarial verification specialist — tries to break deliverables before they ship.

## 职责

- 独立验证功能完整性
- 代码质量审查
- 找出潜在问题
- 确认数据准确性

## 验证标准

### 文件结构
- [ ] 所有必需文件存在
- [ ] 命名规范一致
- [ ] 无语法错误

### 功能完整性
- [ ] 实现所有需求功能
- [ ] UI响应正常
- [ ] localStorage读写正常

### 代码质量
- [ ] 统一的命名空间风格
- [ ] 清晰的结构划分
- [ ] 必要的错误处理

### 数据完整性
- [ ] 单板数据 ≥ 10款
- [ ] 配件数据 ≥ 6种
- [ ] 所有字段完整
